Inline LaTeX

Inline LaTeX is used to embed mathematical expressions within text. Here are examples of the concepts discussed:

  • Big O Notation: The time complexity of an algorithm can be represented as O(n)O(n)O(n) for linear time, O(n2)O(n^2)O(n2) for quadratic time, O(log⁡n)O(\log n)O(logn) for logarithmic time, and O(nlog⁡n)O(n \log n)O(nlogn) for linearithmic time.

  • Turing Machines: The transition function of a Turing machine can be written as δ:Q×Γ→Q×Γ×{L,R}\delta: Q \times \Gamma \rightarrow Q \times \Gamma \times \{L, R\}δ:Q×Γ→Q×Γ×{L,R}, where QQQ is the set of states, Γ\GammaΓ is the tape alphabet, and LLL and RRR represent left and right movement of the read/write head.

  • P vs NP Problem: The P vs NP problem can be stated as PNPP \stackrel{?}{=} NPP=?NP, where PPP is the set of decision problems solvable in polynomial time and NPNPNP is the set of decision problems whose solutions can be verified in polynomial time.

  • Complexity Classes: The complexity classes can be represented as P={L∣∃ a deterministic TM M such that L(M)=L and M runs in time O(nk) for some k}P = \{L \mid \exists \text{ a deterministic TM } M \text{ such that } L(M) = L \text{ and } M \text{ runs in time } O(n^k) \text{ for some } k\}P={L∣∃ a deterministic TM M such that L(M)=L and M runs in time O(nk) for some k}, NP={L∣∃ a nondeterministic TM M such that L(M)=L and M runs in time O(nk) for some k}NP = \{L \mid \exists \text{ a nondeterministic TM } M \text{ such that } L(M) = L \text{ and } M \text{ runs in time } O(n^k) \text{ for some } k\}NP={L∣∃ a nondeterministic TM M such that L(M)=L and M runs in time O(nk) for some k}, PSPACE={L∣∃ a deterministic TM M such that L(M)=L and M uses space O(nk) for some k}PSPACE = \{L \mid \exists \text{ a deterministic TM } M \text{ such that } L(M) = L \text{ and } M \text{ uses space } O(n^k) \text{ for some } k\}PSPACE={L∣∃ a deterministic TM M such that L(M)=L and M uses space O(nk) for some k}, and EXPTIME={L∣∃ a deterministic TM M such that L(M)=L and M runs in time O(2nk) for some k}EXPTIME = \{L \mid \exists \text{ a deterministic TM } M \text{ such that } L(M) = L \text{ and } M \text{ runs in time } O(2^{n^k}) \text{ for some } k\}EXPTIME={L∣∃ a deterministic TM M such that L(M)=L and M runs in time O(2nk) for some k}.
