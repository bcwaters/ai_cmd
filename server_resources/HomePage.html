This servers up docs generated by ai_cmd. This is a simple express server that uses grok 2, give it a second to generate the doc. When set to treeMode, it branches out on each subject in the response, and then returns a consolidated html.

Try a prompt like: "What are the benefits of static html pages"

or "What are some data structures in JS"

or "tell me about applied linear algebra in machine learning"

this is way better in terminal mode, clone it and use it.
